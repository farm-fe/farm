# @farmfe/plugin-react
支持 `React Jsx` 和 `React Refresh`

## Installation

<PackageManagerTabs command="pnpm add @farmfe/plugin-react" />

## Usage
`@farmfe/plugin-react` 是一个 Rust 插件，你只需要在 `farm.config.ts` 的 `plugins` 字段中配置它的包名。
```ts {4}
import { UserConfig } from '@farmfe/core';

const config: UserConfig = {
  plugins: ['@farmfe/plugin-react', { /** options here */}]
}
```

## Options
请参阅[SWC 转换 React 选项](https://swc.rs/docs/configuration/compilation#jsctransformreact)。
