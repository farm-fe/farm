Add `@farmfe/plugin-vue` — a Farm Rust plugin to compile Vue 3 Single-File Components, powered by the [`fervid`](https://github.com/phoenix-ru/fervid) compiler. Provides `.vue` loading, scoped CSS, `v-bind()` CSS variables, custom-element compilation via `.ce.vue` or `features.customElement`, automatic `__VUE_*__` define-flag injection, and `<style lang="…">` virtual sub-modules that hand off to downstream CSS preprocessor plugins.
